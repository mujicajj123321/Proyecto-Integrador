Objetivos

1. Creación de sistema que tome datos relacionados al suelo.

2. Creación de una página que capture datos ingresados por el usuario para mandarlos a la IA por medio de APIs para su analisis y poder mostrarlo al usuario final.

3. Ayudar a los agricultores a mejorar sus cultivos usando IA y agricultura de precisión.

Con el fin de que los agricultores pueden maximizar su producción a partir del uso de nuevas técnicas de agricultura que puedan adquirir a través de las IA, esto a partir de datos como lo son: El clima, el área de cultivo, el tipo de cultivo, etc. Para que la IA pueda realizar recomendaciones teniendo en cuenta todas las variables anteriormente mencionadas.

Hecho por: Andrés Parra y Joaquín Mujica.
